# xtermvm
xTerm installator for Proxmox VMs

# Usage
Method 1
``` sh
git clone https://github.com/GDLev/xtermvm
cd xtermvm
sh xtermvm.sh
```
Method 2
``` sh
wget cdn.gdlev.ml/xtermvm.sh 
sh xtermvm.sh
```
Method 3
``` sh
curl https://raw.githubusercontent.com/GDLev/xtermvm/main/xtermvm.sh | bash
```
